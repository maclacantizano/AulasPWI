Atributos Função
id Define um identificador exclusivo (ID) que deve ser único por todo o documento,Seu objetivo é identificar o elemento ao navegar por âncoras
style contém informações de estilo para um documento ou uma parte do documento. As informações de estilo específico estão contidas dentro deste elemento, geralmente no CSS.
class é uma lista das classes de um elemento, separada por espaços.
